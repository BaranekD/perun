How Perun helps project managers

Users management
Create customizable registration forms to your organization
Manually or automatically approve user's applications
Set up membership expiration
Assign additional information to the user entry in the Perun
Groups management
Organize users into groups
Create applications to groups in your organizations
or delagates rights to manage group members
Resource management
Allow groups of users to utilize resources
Set individual configuration for each user if you want to
Manage SLD (service level definition) for each resource
